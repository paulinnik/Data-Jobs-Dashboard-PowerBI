# Data Jobs Dashboard — Аналитика рынка вакансий в Power BI

В этом репозитории два интерактивных **Power BI дашборда**, которые помогают быстро разобраться в рынке data-вакансий за 2024 год.

Данные взяты из открытого источника: [datanerd.tech](https://datanerd.tech/).

## Сравнение двух версий дашбордов

| Характеристика              | **Version 1** (Data Jobs Dashboard)                                | **Version 2.0** (Data Jobs Dashboard 2.0)                                       |
| --------------------------- | ------------------------------------------------------------------ | ------------------------------------------------------------------------------- |
| **Количество страниц**      | 2 страницы                                                         | 1 компактная страница                                                           |
| **Глубина анализа**         | Более детальная                                                    | Быстрый обзор                                                                   |
| **Количество визуализаций** | Больше графиков и представлений                                    | Меньше, но они динамические                                                     |
| **Ключевые фичи**           | Drill-through, Buttons & Bookmarks, отдельная страница детализации | Field Parameters (динамическая смена метрик), улучшенный дизайн и модель данных |
| **Для кого лучше подходит** | Тем, кто хочет глубоко погрузиться в конкретную должность          | Тем, кому нужен быстрый и удобный обзор рынка                                   |
| **Файл**                    | `Data_Jobs_Dashboard.pbix`                                         | `Data_Jobs_Dashboard_2.0.pbix`                                                  |

## Data Jobs Dashboard (Version 1)

**Многостраничный дашборд** с глубоким анализом.

### Что внутри:

- **Page 1: High-Level Market View** — обзор рынка: общее количество вакансий, медианная зарплата, топ позиций.
- **Page 2: Job Title Drill Through** — детализация по конкретной должности.

**Продемонстрированные навыки:**

- Implicit Measures
- Разные типы графиков (Column, Bar, Line, Area)
- Drill-through, Buttons & Bookmarks, Slicers
- Продуманный дизайн и навигация

**Файл:** [`Data_Jobs_Dashboard.pbix`](Data_Jobs_Dashboard.pbix)

[Подробное описание и скриншоты →](./Project%201/README.md)

## Data Jobs Dashboard 2.0

**Одностраничная версия** с акцентом на удобство и динамику.

### Что внутри:

- Всё самое важное на одной странице: ключевые KPI, популярность навыков, сравнение зарплат.
- Два интерактивных графика с **Field Parameters**:
  - Skill Popularity → переключение между % вакансий и количеством
  - Job Salaries → переключение между Yearly и Hourly зарплатой
- Улучшенная модель данных (Star Schema) и Power Query ETL.

**Продемонстрированные навыки:**

- Field Parameters
- Star Schema и качественный Data Modeling
- DAX-меры
- Чистый, минималистичный дизайн

**Файл:** [`Data_Jobs_Dashboard_2.0.pbix`](Data_Jobs_Dashboard_2.0.pbix)

[Подробное описание и скриншоты →](./Project%202/README.md)

## Для кого эти дашборды

- Соискателям data-позиций
- Людям, которые меняют направление карьеры
- Всем, кто хочет быстро понять уровень зарплат, востребованность навыков и географию рынка в 2024 году

Оба дашборда полностью интерактивны — можно фильтровать по должности, стране и другим параметрам.
